GeoGuide AI is an intelligent, location-aware web application that helps users instantly discover places, services, and geographic details anywhere in the world. Powered by AI-driven natural language search and real-time map data, the platform makes exploring any area fast, intuitive, and effortless.

This repository contains the full source code for the GeoGuide AI app built using AI Studio, leveraging modern web technologies and map APIs.
